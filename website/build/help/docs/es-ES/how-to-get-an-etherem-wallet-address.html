Cómo obtener una wallet de Ethereum

MetaMask is one of the simplest ways to get and manage an Ethereum address. it is a free to use browser extension, you can read more about it and how to install it in their getting started guide.

If you want to get a bit more serious about security, a hardware wallet is a great option. Ledger makes a great line of hardware wallets that you can use with the Colony dapp. You can check out their range on the Ledger Store.
